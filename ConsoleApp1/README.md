## Running Tests

This project uses the **NUnit** testing framework.

### Requirements
Before running the tests, make sure the following NuGet packages are installed in your test project:
- `NUnit`
- `NUnit3TestAdapter`
- `Microsoft.NET.Test.Sdk`

You can install them using the command line:
```bash
dotnet add package NUnit
dotnet add package NUnit3TestAdapter
dotnet add package Microsoft.NET.Test.Sdk
